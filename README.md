MFCMVVM - User Directory App

A modern Android app built using MVVM architecture that fetches users from an API, caches data locally, and supports offline usage.

🚀 Features
MVVM Architecture
API integration (Retrofit, OkHttp)
Offline support (Room Database)
Image loading (Glide)
Shimmer loading effect
Search functionality
Swipe-to-refresh
Error handling & retry
🖼 Screenshots
List View	Search & Offline

	
🛠 Tech Stack
Kotlin
MVVM
Room, Retrofit, Glide
Coroutines + LiveData/Flow

Clone the repo
Open in Android Studio
Run the app
